# 跨界工藝：台灣長鬃山羊與印尼Batik的互動古鎖模擬器
> **大學生網頁互動工藝與文化符號之跨界專題研究** [![GitHub Pages](https://img.shields.io/badge/Live_Demo-GitHub_Pages-blue?style=for-the-badge&logo=github)](https://[su2006sgm].github.io/[-]/)
![Tech Stack](https://img.shields.io/badge/Tech_Stack-HTML5%20%7C%20TailwindCSS%20%7C%20VanillaJS-orange?style=for-the-badge)

本專案是一款基於網頁原生技術（HTML5, Tailwind CSS, Vanilla JavaScript）開發的**跨文化互動工藝模擬器**。專案核心將**台灣特有種長鬃山羊**在絕壁攀登的堅韌意象，與**印尼西爪哇東巴加魯特羊（Domba Garut）**的強悍榮耀精神進行多維度縫合，並以**印尼皇家蠟染圖騰（Batik Parang）**作為3D表面材質貼圖（PBR）的創新嘗試，最終載體呈現為一把具備精密解鎖機關的傳統中式簧片古鎖。

觀看者可以透過此線上模擬器，以數位互動方式深度探索東南亞工藝歷史、機械力學結構與豐富的生態美學。

## 💡 設計靈感與文化溯源

### 1. 學術脈絡：南島語族遷徙 (Austronesian Migration)
本專案的跨國文化結合並非隨機拼湊，而是奠基於語言學與考古學著名的「出台灣假說」（Out of Taiwan）。專案以此歷史遷徙路徑為背景，透過數位工藝讓南島語族的起點（台灣）與重要節點（印尼）進行一場跨越時空的藝術對話。

### 2. 生態意象轉譯：雙羊對比
* **台灣長鬃山羊 (Taiwan Serow)**：特有種，生性孤傲、具強大攀登能力，其特化蹄腳能於裸岩峭壁如履平地。在作品中作為古鎖的視覺主體結構，象徵「在絕境中攀向高峰」的堅韌。
* **東巴加魯特羊 (Domba Garut)**：印尼西爪哇著名綿羊，具有巨大向後彎曲的粗壯犄角，在當地傳統鬥羊競技中象徵無畏的戰鬥力與尊貴地位。本專案將其強悍的內在精神灌注於古鎖中。

### 3. 材質創新：Batik 蠟染圖騰鑲嵌
捨棄常規的寫實動物毛皮材質，改採印尼傳統皇家專用的 **Batik Parang（破浪紋）** 與 **Batik Kawung（棕櫚十字紋）**。Parang 圖騰其S型幾何線條描繪「海浪拍擊懸崖」的景象，在視覺意象上與台灣長鬃山羊棲息的險峻峭壁完美扣合。

## 🛠️ 技術亮點與核心功能

本模擬器完全採用**原生 Web 技術**打造，無需下載任何插件，並針對以下前端技術進行深度實作：

* **3D 視差旋轉運算 (Parallax View)**：利用滑鼠監聽事件即時動態計算 SVG 矩陣（Translate & Scale），在二維網頁上呈現出極具層次感的立體 3D 視覺深淺效果。
* **中式簧片鎖機構模擬 (Splitting Spring Mechanism)**：嚴格考究傳統亞洲古鎖力學，在網頁中實作「X-Ray 內部透視模式」。觀看者可拖曳鑰匙，即時檢視鑰匙凹槽如何擠壓收攏「黃銅V型簧片」，完成物理結構上的解鎖邏輯。
* **自研 Web Audio API 聲學沉浸**：不使用外掛音訊檔，完全透過純 Web Audio API 程式碼合成金屬彈片摩擦（Spring Scrape）、鎖樑拖曳（Shackle Slide）的清脆操作音，並合成逼真的羊群環境背景音（Ambient Sound）。
* **自主行為 AI 虛擬桌寵系統**：畫面下方搭載一隻桌寵小羊（Desktop Pet）。透過自訂行為腳本，小羊會自主尋找草地、奔跑與進食。開發上利用「外層座標位移」與「內層節點獨立旋轉」的雙重參數控制，**完美解決了網頁 SVG 動態拉伸時常見的骨骼解體與變形問題**。
* **客製化工坊 (Workshop UI)**：支援青銅（Aged Bronze）、皇室黃金（Royal Gold）、生鐵黑（Forged Iron）等 PBR 材質即時切換，並內建動態微振動回饋（Haptics API）與多國語系介面切換（繁中 / 印尼文）。
